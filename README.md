# CleanPaste & SEO Formatter ✍️

A specialized productivity tool for SEO writers, bloggers, and data entry specialists. This **AutoHotkey v2.0** script helps you maintain a clean workflow by stripping messy formatting and optimizing text cases for search engine headlines.

## ✨ Key Features
- **Paste as Plain Text:** Instantly strips rich text formatting (fonts, colors, links) from your clipboard before pasting.
- **Title Case Converter:** Automatically converts any text to "Title Case" (Every Word Capitalized), perfect for SEO headings and blog titles.
- **Focus-Safe UI:** Uses advanced GUI properties so you don't lose focus on your active writing window.

## ⚙️ Installation
1. Install [AutoHotkey v2.0](https://www.autohotkey.com/).
2. Download `CleanPasteFormatter.ahk` from this repository.
3. Run the script and use the floating GUI to format your text.
